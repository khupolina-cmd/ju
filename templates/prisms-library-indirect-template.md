# Библиотека призм: Косвенные конкуренты

> **Сюда вставляй ТОЛЬКО косвенные конкуренты** — другой тип продукта или формат, но пересекающаяся потребность или аудитория.  
> Примеры: лосьон для тела рядом с кремом для рук, духи рядом с парфюмированным кремом.  
> Каждый блок — это результат Skill 2 по конкуренту (Секция 7 вывода).  
> Для прямых конкурентов используй `prisms-library-direct.md`.

---

## Как заполнять

1. Запусти Skill 2 по конкуренту
2. Возьми Секцию 7 из вывода (компактный блок для библиотеки)
3. Вставь блок под горизонтальной чертой ниже
4. Повтори для каждого косвенного конкурента

Структура каждого блока:

```
---
product_key: [артикул или название]
category: косвенный конкурент
[КЛАСТЕРЫ, CONFLICT_MAP, КАЛИБРОВКА — из Skill 2]
```

---

[вставляй блоки косвенных конкурентов сюда]
