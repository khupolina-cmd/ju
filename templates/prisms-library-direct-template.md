# Библиотека призм: Прямые конкуренты

> **Сюда вставляй ТОЛЬКО прямые конкуренты** — тот же тип продукта, та же категория, конкурируют за одного покупателя.  
> Каждый блок — это результат Skill 2 по конкуренту (Секция 7 вывода).  
> Для косвенных конкурентов используй `prisms-library-indirect.md`.

---

## Как заполнять

1. Запусти Skill 2 по конкуренту
2. Возьми Секцию 7 из вывода (компактный блок для библиотеки)
3. Вставь блок под горизонтальной чертой ниже
4. Повтори для каждого прямого конкурента

Структура каждого блока:

```
---
product_key: [артикул или название]
category: прямой конкурент
[КЛАСТЕРЫ, CONFLICT_MAP, КАЛИБРОВКА — из Skill 2]
```

---

[вставляй блоки прямых конкурентов сюда]
